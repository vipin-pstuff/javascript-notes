## difference b/w if else if statement & if if statements ğŸ”¥

    - checking for multiple condition 
        then we use if else if else statements & if if statements ğŸ’¡ğŸ’¡ğŸ’¡

    -> if else if else statement âœ…

        if (totalPrice == null) {
            console.log("there is no total price")
        } else if (totalPrice < 10) {
            console.log("there is pretty cheap")
        } else if (totalPrice < 50) {
            console.log("there is too bad")
        } else {
            console.log("there is expensive")
        }

        NOTE : in if else if else statement ğŸ“

            - important note ğŸ”¥

            - in if else if else statement , 
                if the condition of anyone statement is true
                then that will be executed ğŸ’¡ğŸ’¡ğŸ’¡
            - & after that statement (which is true) 
                then all below -> else if statements will be skipped from checking
                even if those below -> else if statements are true in terms of condition ğŸ’¡ğŸ’¡ğŸ’¡  
     
        // output : this is pretty cheap

        - that's why here we got only one output 
            even though next else if statement is true in terms of condition ğŸ’¡ğŸ’¡ğŸ’¡

    -> if if statements âœ…

        if (totalPrice == null) {
            console.log("there is no total price")
        } if (totalPrice < 10) {
            console.log("there is pretty cheap")
        } if (totalPrice < 50) {
            console.log("there is too bad")
        } else {
            console.log("there is expensive")
        }

        NOTE : in if if statements ğŸ“

            - important note ğŸ”¥

            - in if if statements , 
                if anyone statement is true in condition
                then that statement will be executed ğŸ’¡ğŸ’¡ğŸ’¡
            - but other if statements are true in terms of condition 
                which are below that if statement (which is true) ğŸ’¡ğŸ’¡ğŸ’¡
            - then those other if statements will executed
                because all the if if statements checked ğŸ’¡ğŸ’¡ğŸ’¡

        - that's why output is 
        // output : there is pretty cheap
                    there is too bad

    NOTE : important note - about if if statements and if else if statementsğŸ”¥

        - these flow happens differently in both if else if else statement & if if statement
            because ğŸ’¡ğŸ’¡

        - in if & if statements , 
            each & every statements are separated/different from each other
            that's why they don't impact on each other ğŸ’¡ğŸ’¡ğŸ’¡
        
        - in if else if statements , 
            each & every if else if statements 
            are same & related to each other
            that's why they have impact on each other ğŸ’¡ğŸ’¡ğŸ’¡
